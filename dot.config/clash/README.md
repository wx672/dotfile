- To start the clash proxy

	```sh
	chmod +x clash-linux-amd64-v1.6.5
	./clash -f config.yml
	```

- Proxy settings
  - socks5: `socks://localhost:7891`
  - http: `http://localhost:7890`
  - https: `https://localhost:7890`
  - `/etc/proxychains4.conf`: `socks5	127.0.0.1 7891`
